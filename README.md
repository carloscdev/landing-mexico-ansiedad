## Running the app

* __Clonar el proyecto__
```bash
$ git clone https://github.com/carloscdev/landing-mexico-ansiedad
```

* __Instalar dependencias__
```bash
$ yarn install
```

* __Ejecutar la aplicación en desarrollo__
```bash
$ yarn dev
```
